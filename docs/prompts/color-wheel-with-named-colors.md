# Color Wheel with Named Colors

!!! prompt
    Use the microsim-p5 skill to create a new microsim called 'color-wheel-with-named-colors'.
    In this microsim, the user is presented with a color wheel in the drawing area.
    On the color wheel are a series of dots.  Each dot represented one of the web-safe
    named colors.  When the user hovers over a dot, the name of the color and the RGB
    values appear in the control region.
